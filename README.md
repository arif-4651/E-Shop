# Professional E-commerce Frontend

A modern, professional e-commerce frontend built with HTML, CSS, and JavaScript.

## Features

- ✅ **Multiple Pages**: Home, Products, Cart, Checkout, Login, Signup, Profile
- ✅ **Professional Design**: Modern UI/UX with clean aesthetics
- ✅ **Responsive Design**: Works on mobile, tablet, and desktop
- ✅ **Shopping Cart**: Add, remove, and update cart items
- ✅ **Form Validation**: Login, signup, and checkout forms
- ✅ **Product Filtering**: Filter products by category
- ✅ **Local Storage**: Cart data persists across sessions
- ✅ **Bangladeshi Products**: Includes traditional Bangladeshi products

## Project Structure

```
E-commerce/
├── images/              # All product and UI images
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── main.js         # JavaScript functionality
├── pages/
│   ├── login.html      # Login page
│   ├── signup.html     # Signup page
│   ├── products.html   # Products listing page
│   ├── cart.html       # Shopping cart page
│   ├── checkout.html   # Checkout page
│   └── profile.html     # User profile page
└── index.html          # Home page
```

## Pages

1. **Home Page** (`index.html`)
   - Hero section
   - Featured products
   - Category sections

2. **Login Page** (`pages/login.html`)
   - Email and password login
   - Form validation
   - Remember me option

3. **Signup Page** (`pages/signup.html`)
   - User registration form
   - Password confirmation
   - Terms & conditions

4. **Products Page** (`pages/products.html`)
   - All products grid
   - Category filtering
   - Add to cart functionality

5. **Cart Page** (`pages/cart.html`)
   - View cart items
   - Update quantities
   - Remove items
   - Order summary

6. **Checkout Page** (`pages/checkout.html`)
   - Shipping information form
   - Payment method selection
   - Order summary

7. **Profile Page** (`pages/profile.html`)
   - Personal information
   - Address management
   - Order history

## How to Use

1. Open `index.html` in a web browser
2. Navigate through different pages using the navigation bar
3. Add products to cart
4. Proceed to checkout
5. Login/Signup to access profile

## Technologies Used

- **HTML5**: Structure
- **CSS3**: Styling with modern features
- **JavaScript**: Functionality and interactivity
- **Local Storage**: Cart persistence

## Design System

- **Primary Color**: Indigo (#6366F1)
- **Font**: Inter (Google Fonts)
- **Spacing**: 8px multiples
- **Shadows**: Subtle elevation system
- **Border Radius**: 6px, 8px, 12px
